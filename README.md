# CSV データ可視化 Webアプリ

## 概要
本ツールは、ブラウザ上でCSVファイルをアップロードし、その内容をインタラクティブな表形式で閲覧できるFlaskベースのWebアプリケーションです。

「非エンジニアでも、ブラウザから簡単にCSVの中身を確認・共有できる」ことを目的としています。

## ディレクトリ構成
```text
csv-web-app/
├── templates/          # HTMLテンプレート
│   └── index.html      # アップロード画面
├── venv/               # 仮想環境（Python 3.11）
├── .gitignore          # Git除外設定
├── app.py              # Flaskメインプログラム
├── requirements.txt    # 依存ライブラリ一覧（一括インストール用）
└── README.md           # 本ファイル（このドキュメント）
```

## 機能
- ローカルのCSVファイルをブラウザ経由でサーバーへ送信
- データ一覧表示: Pandas を使用し、CSVを構造的なHTMLテーブルとしてレンダリング
- セキュリティ配慮: `secure_filename` により、サーバーを攻撃から守るためのファイル名変換（サニタイズ）を実装
- レスポンシブUI: スマートフォンやPCなど、閲覧環境に合わせて表のサイズを自動調整

## 動作内容
1. ブラウザでアプリ（localhost:5000）にアクセス
2. CSVファイルを選択して「アップロード」ボタンをクリック
3. app.py がファイルを受け取り、Pandasで読み込み
4. HTMLテンプレートへデータを渡し、ブラウザ上に表を表示

## 実行方法

### セットアップ方法
### 1. リポジトリをクローンまたはダウンロード
プロジェクトのフォルダに移動します。 
```bash
cd csv-web-app
```

### 2. 仮想環境の作成と起動 (推奨)
```bash
python -m venv venv
source venv/bin/activate  # Macの場合
# .\venv\Scripts\activate  # Windowsの場合
```
### 3. 必要なライブラリを一括インストール
```bash
pip install -r requirements.txt
```
### 4. アプリを起動
```bash
python app.py
```
### 5. ブラウザでアクセス
http://127.0.0.1:5000 にアクセスしてください

## 想定される利用シーン
- コマンドライン操作に慣れていないユーザーへのデータ提供
- CSVデータの中身を手軽にブラウザで確認
- 社内向け簡易データ管理ツールのプロトタイプ

## 今後の拡張予定
- グラフ表示機能: 特定のカラムをグラフ化（Matplotlib/Plotly）
- フィルタ・並び替え機能: ブラウザ上でデータの絞り込み
- ログイン機能: 特定のユーザーのみ閲覧可能にする
- データベース連携: アップロードしたデータをSQLite等に保存

## 使用技術
- **Language:** Python 3.11 (venvにより環境を分離)
- **Framework:** Flask 3.0.3
- **Library:** - Pandas 2.0.3 (CSV解析)
- **Frontend:** HTML5 / CSS3
